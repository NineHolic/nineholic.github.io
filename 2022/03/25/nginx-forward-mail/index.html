内网邮件通过 Nginx 代理进行转发

1、安装 stream

使用 stream 方式代理 smtp 协议的邮件服务，需安装 stream 模块

# 查看已安装模块
./nginx -V

# 编译时需加上 stream 模块
./configure --prefix=/usr/local/nginx --with-stream ...
make && make install
2、nginx 配置
stream {
    # 邮件转发
    server {
        listen 8765;
        proxy_connect_timeout 5s;
        proxy_timeout 5s;
        proxy_pass smtp.qq.com:587;
    }
}
